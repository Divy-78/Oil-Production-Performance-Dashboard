# Oil Production Performance Dashboard

## Tools Used
- Power BI
- SQL
- DAX

## Dashboard Features
- Total Oil Production
- Total Gas Production
- Total Water Production
- Average GOR
- Average Water Cut
- Oil Production Trend
- Downtime by Well
- Oil Production by Well
- Oil Production by Field
- Field and Date Filters

## Dataset
Daily production data from oil wells.
